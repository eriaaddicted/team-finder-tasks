# Team Finder

веб-платформа для поиска единомышленников, создания команд и совместной работы над ИТ-проектами. 

### Использованные технологии:
- Python 3.13 / Django 5.2 
- PostgreSQL / Docker Compose

### Локальный запуск:
1. Клонирование репозитория и настройка окружения
    Склонируйте репозиторий с исходным кодом проекта на свой локальный компьютер:
      git clone <ссылка_на_ваш_репозиторий>
2. После успешного скачивания перейдите в корневую директорию проекта:
    cd team-finder-ad-main-3
3. Настройка переменных окружения (.env)
  Проект использует переменные окружения для защиты конфиденциальных данных и гибкой настройки подключения.
    Создайте копию файла .env_example. и назовите её .env
4. Откройте созданный файл .env в любом текстовом редакторе(VS Codeб например),астройте блок подключения к базе данных PostgreSQL. Для запуска Django локально на вашем компьютере , параметры должны быть следующими:


    # Секретный ключ Django
    SECRET_KEY=django-insecure-your-super-secret-key

    # Режим отладки
    DEBUG=True

    # Настройки хостов
    DJANGO_ALLOWED_HOSTS=127.0.0.1,localhost

    # Параметры подключения к PostgreSQL
    POSTGRES_DB=team_finder
    POSTGRES_USER=team_finder
    POSTGRES_PASSWORD=team_finder
    POSTGRES_HOST=127.0.0.1
    POSTGRES_PORT=5432

    # Системная версия таски
    TASK_VERSION=2
